### 🧱 Arquitetura e Ecossistema
* **Kernel vs. Shell:** Compreensão das diferenças arquitetónicas e práticas entre o núcleo do sistema (Kernel) e a interface de comandos (Shell).
* **Distribuições Linux:** Análise do vasto ecossistema de distribuições (Distros) e identificação das principais opções exigidas pelo mercado corporativo atual.

### 🗂️ File System Hierarchy (FHS)
Domínio da estrutura padrão de diretórios utilizada pelos sistemas Linux, fundamental para administração, troubleshooting e segurança:

| Diretório | Descrição |
| :--- | :--- |
| `/` | Diretório raiz do sistema (Root). |
| `/bin` | Comandos e executáveis essenciais. |
| `/boot` | Ficheiros necessários para a inicialização do sistema. |
| `/dev` | Representação dos dispositivos de hardware. |
| `/etc` | Ficheiros de configuração do sistema. |
| `/home` | Diretórios pessoais dos utilizadores. |
| `/lib` | Bibliotecas utilizadas pelos programas e pelo sistema. |
| `/media` | Ponto de montagem para mídias removíveis. |
| `/mnt` | Montagem temporária de dispositivos e sistemas de ficheiros. |
| `/opt` | Instalação de softwares e pacotes opcionais. |
| `/root` | Diretório pessoal do utilizador root. |
| `/sbin` | Ferramentas administrativas do sistema. |
| `/srv` | Dados utilizados por serviços do sistema. |
| `/tmp` | Ficheiros temporários. |
| `/usr` | Aplicações e recursos voltados aos utilizadores. |
| `/var` | Dados variáveis, como logs, cache e ficheiros temporários. |
